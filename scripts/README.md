# Metadata And Utility Scripts

Utilities for building and auditing the Waymo front-camera metadata tables.

- `extract_e2e_rich_metadata_shards_v2_checked.py`: extracts rich per-frame metadata shards.
- `build_e2e_clean_tables.py`: builds cleaned frame and scenario tables.
- `build_e2e_window_manifests.py`: builds fixed-frame window manifests.
- `audit_e2e_metadata_schema.py`: audits schema consistency.
- `preprocess_e2e_front.py`: preprocessing helper for front-camera assets.
